Sign Up Page 입니다
프로젝트 내에서의 html 파일 경로 : templates/account/sign-up.html
이 페이지는 주소표시줄에 localhost:8080/sign-up 이라고 입력하면 표시됩니다
/sign-up <-- localhost:8080/sign-up 과 같은 의미입니다
주소표시줄에 localhost:8080/sign-up 이라고 입력하면
SpringFramework 가 @GetMapping("/sign-up") 을 인식해서
AccountController 클래스의 signUpForm(Model model) 메소드가 자동으로 호출됩니다 
  @GetMapping("/sign-up")
  public String signUpForm(Model model){
    return "account/sign-up";
  }

signUpForm(Model model) 메소드의 return 값
return "account/sign-up" 의 의미는
localhost:8080/sign-up 요청이 들어오면
templates/account/sign-up.html 파일을 보여준다는 의미입니다
